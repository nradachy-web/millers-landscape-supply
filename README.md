# Miller's Landscape & Supply

Marketing site for Miller's Landscape & Supply — New Hudson, MI.

Live: [millerslandscapesupplies.com](https://millerslandscapesupplies.com/)

## Stack

Static HTML / CSS / vanilla JS, hosted on GitHub Pages with a CNAME on Namecheap DNS.

## Local dev

```sh
python3 -m http.server 8765
open http://localhost:8765/
```

## Deploy

Push to `main`. GitHub Pages serves the root.
